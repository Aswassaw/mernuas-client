# Mernuas
Client untuk Mernuas, sebuah aplikasi berbasis MERN Stack dengan studi kasus sebuah authentikasi luar biasa, yaitu login dengan Google dan/atau Github.

## Teknologi yang digunakan
- ReactJS
- Redux
- React Toastify
- Framer Motion
- React Github Login
- React Google Login
- Netlify
